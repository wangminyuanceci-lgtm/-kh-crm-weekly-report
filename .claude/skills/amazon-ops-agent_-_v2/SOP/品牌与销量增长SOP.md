# 品牌与销量增长 SOP

> 适用场景：广告基础健康，目标是冲销量增长 / 建品牌竞争度
> 配套：[投手周复盘SOP.md](投手周复盘SOP.md)（日常）+ [排名下降应对SOP.md](排名下降应对SOP.md)（异常应急）+ 本 SOP（战略增长）

---

## 🎯 核心认知 — 区分两个目标

| 目标 | 想要的 | 看哪个指标 | 战术偏向 | 时长 |
|---|---|---|---|---|
| **提升销量** | 短期单数增加 / GMV 增长 | 销售额增速 / BSR / Sessions | 抢量 + Coupon + 主力加投 | 1-3 月可见效 |
| **提升品牌竞争度** | 品类心智 / 长期份额 | 品牌词搜索量 / 品类核心词覆盖 | 曝光 + 评论 + SBV / Brand Story | 3-12 月长期 |

**两者重叠但不同**：
- 销量增长：靠促销/加广告短期堆，但停了就掉
- 品牌竞争度：长期积累，起来后自带流量，未来 TACOS 自然下降

**理想节奏**：
```
短期推销量 (拉单数 + 评论)
    ↓
中期建品牌 (SBV + 站外)
    ↓
品牌带自然流量 → TACOS 下降
    ↓
省下的钱投新品（复利循环）
```

## 📈 提升销量 — 3 个时间维度

### A. 短期销量（1-2 周见效）— "速效药"

| 战术 | 具体做法 | 速度 | 副作用 |
|---|---|---|---|
| **Coupon 15-25%** | 卖家中心 → Promotions → Coupons | 立刻生效 | 毛利降 5-15pp |
| **Lightning Deal / Deal of the Day** | 后台申请，需达成资格 | 24-48h 流量爆发 | 平台抽成 |
| **加主力 SKU 广告预算 30-50%** | BulkSheet 提 Daily Budget | 1-3 天 | TACOS 上升 |
| **Top of Search Bid+ 提到 +200%** | 抢黄金位置 | 1 天 | CPC 上升 |
| **SB 流量导到 Storefront** | SB Brand Store Link 投放 | 1-3 天 | - |

**适用场景**：
- 本月业绩冲刺
- 库存清仓（高库存 SKU）
- Prime Day / BFCM 等大促

**注意**：短期销量结束后**必然回落**，不能依赖。

### B. 中期销量（1-3 个月）— "增量发动机"

| 战术 | 价值 | 实施要点 |
|---|---|---|
| **扩 SKU 矩阵** | 一父ASIN 多色/尺寸变体覆盖更多需求 | 颜色/尺寸/套装组合 |
| **拓品类长尾词** | 建独立 Exact Campaign | 用关键词分析报告 ⭐ 名单 |
| **跨品类配套销售** | RACKS 配杠铃片 / 哑铃配 rack | 客单价提升 |
| **新增 SD 浏览再营销** | 引导未购买用户回来 | views=(exact-product lookback=30) |
| **Subscribe & Save** | 适合配件/快消（FBA 品线）| 用户绑定订阅 |

### C. 长期销量（3-12 个月）— "复利增长"

| 战术 | 价值 |
|---|---|
| **新品孵化** | 旧品类天花板 → 新品类破局 |
| **站外引流**（KOL / 社媒 / Reddit） | 算法奖励外部流量 |
| **季节性提前布局**（BFCM 8 月就开始备）| 大促 ROI 远高平时 |
| **价格弹性测试** | 提价 5% 看销量降多少 — 找毛利甜点 |
| **Brand Loyalty 计划** | 老客户复购率 |

## 🏆 提升品牌竞争度 — 3 个维度

### 维度 1：品牌曝光（让用户看到你）

| 战术 | 工具/位置 | 重点 |
|---|---|---|
| **SB Video** | Sponsored Brands Video | 抢核心词搜索结果顶部 |
| **SB Store Spotlight** | SB - Brand Store | 引导到 Storefront 沉浸式体验 |
| **SD VCPM 品类曝光** | Sponsored Display - VCPM | 千次曝光定价，品牌覆盖 |
| **Amazon Posts** | Brand Tools → Posts | 免费曝光，类似社交 feed |
| **Brand Story 模块** | Listing 详情页 | 讲品牌故事 |
| **本品防御 SD** | SD ASIN 定向 | 防止自家详情页被竞品占 |

**关键指标**：核心品类词搜索结果首页自家品牌出现频次

### 维度 2：品牌信任（让用户买你）

| 战术 | 怎么做 | 周期 |
|---|---|---|
| **A+ Premium Content** | Listing → A+ Content → Premium | 1-2 周 |
| **Vine 计划送评** | 主力新品 30 单 Vine | 4-8 周见评 |
| **视频/3D 主图** | 提升详情页转化 | 2-4 周 |
| **包装升级** | unboxing 体验 → 好评率 | 1-3 月 |
| **Customer Service** | Buyer Messages 4 小时内回复 | 持续 |

**关键指标**：评分 / 评论数 / 评论关键词正向率

### 维度 3：品牌词流量池建设（**最被低估**）

**品牌健康度核心指标**：搜你品牌名的搜索量增长

| 监控点 | 工具 | 目标 |
|---|---|---|
| **品牌词月搜索量** | Brand Analytics → Search Terms | 月环比增长 5-10% |
| **品牌词排名** | 搜你品牌后自家产品排名 | Top 10 |
| **品牌词 ACOS** | 关键词分析报告 | < 5%（应该极低）|
| **"品牌词 + 品类"长尾**（如 ritfit dumbbells） | 搜索词分析 | 数量增长 = 品牌认知扩散 |

**例**：WEIGHTS 品线 ritfit 品牌词 ROAS 18.6 / ACOS 5.4% — 是品牌建设非常强的信号。

## 📊 关键指标体系切换

### 不同阶段看什么

| 阶段 | 主指标 | 副指标 |
|---|---|---|
| **销量冲刺期**（短期）| 销售额日增速 / BSR | 单位销量速率 / TACOS |
| **品牌建设期**（中长期）| **品牌词搜索量增长** / 品类核心词覆盖率 | SBV 曝光 / 评分 / 自然订单占比 |
| **效率优化期**（成熟期）| TACOS 下降 / ROAS 上升 | 利润率 |

### 健康品牌的 4 个信号

| 信号 | 数据表现 |
|---|---|
| 品牌词搜索量持续增长 | Brand Analytics |
| 自然订单占比 > 30% | 报告"自然订单占比"列 |
| 品牌词 ACOS < 5% | 关键词分析报告 |
| Sessions 中"品牌词流量"占比 > 15% | Brand Analytics |

## 📋 阶段性执行模板

### 模板 A：销量冲刺（如月底冲业绩 / 大促前）

```
目标：本月销售额 +30%
时长：2-4 周

第 1 周（启动）
- [ ] 主力 Top 5 SKU 设 Coupon 15-20%
- [ ] 主力 Top 3 SKU 申请 Lightning Deal
- [ ] 加这些 SKU 的 SP 预算 30-50%
- [ ] Top of Search Bid Modifier 提到 +200%
- [ ] 监控库存确保 > 60d

第 2-3 周（持续）
- [ ] 跟踪日销售额增速
- [ ] 跟踪 BSR 变化
- [ ] TACOS 容忍升到 15-18%
- [ ] 若 ROI 不行调整 Coupon 比例

第 4 周（收尾）
- [ ] 销售额达标后逐步恢复 Coupon
- [ ] 把 Top of Search Bid+ 降回 +100%
- [ ] 评估"投入产出" — 这次冲刺值不值
```

### 模板 B：品牌竞争度建设（季度级）

```
目标：3 个月内品牌词搜索量增长 30%
时长：3-6 个月

第 1 月（基础设施）
- [ ] Brand Store 完善（如未做）
- [ ] A+ Content 升级（如未做）
- [ ] Brand Story 模块上线
- [ ] 主力 5 SKU 启动 Vine 计划

第 2 月（曝光建设）
- [ ] 启动 SB Video 投核心品类词
- [ ] 启动 SD VCPM 品类覆盖
- [ ] Amazon Posts 每周 3 帖
- [ ] 评论持续维护（Request a Review）

第 3 月（评估 + 强化）
- [ ] Brand Analytics 看品牌词搜索量增长
- [ ] 看品牌词 ACOS 是否进一步下降
- [ ] 看自然订单占比是否上升
- [ ] 强化表现好的 SBV / SD 投放
```

### 模板 C：销量 + 品牌双线（推主力新品 / 重新做新品类）

```
目标：3 个月做出 1 个 BSR Top 100 新主力
时长：3-6 个月

启动期（1-4 周）
- [ ] 产品验证（评分 ≥ 4.0 / 评论 ≥ 30）
- [ ] Vine 计划送评
- [ ] 低 ACOS 容忍度（前期 30-50% 接受）
- [ ] 主推 SP Exact + SP Auto

冲刺期（4-12 周）
- [ ] 加广告预算到日均 $50-100
- [ ] Coupon 20% 持续
- [ ] 启动 SB Video 抢核心词
- [ ] 跟踪 BSR 进入 Top 1000 → Top 500 → Top 100

收成期（12 周后）
- [ ] BSR Top 100 后逐步降 Coupon
- [ ] TACOS 从 30% → 15% 回归
- [ ] 转入"品牌建设期"维护
```

## 💡 经验法则

### 1. 先有评论再加投

新品 Vine 30 评 + 4.0+ 评分前**不要大额加广告** — 评分差时引流转化必然低。

### 2. **品牌词搜索量 > 任何其他指标**

如果只看 1 个指标判断品牌健康，就看 Brand Analytics 里**品牌词月搜索量趋势**。
- 增长 = 品牌力增强
- 持平 = 没在建品牌
- 下降 = 品牌力流失（比短期销量更可怕）

### 3. 销量提升 ≠ 品牌建设

- Coupon 拉销量但不建品牌（用户买完不记得你）
- SB Video 建品牌但不一定立刻出单（曝光价值滞后）
- **两者要并行做** — 不能只追一个

### 4. **不要在产品力不足时建品牌**

如果评分 < 4.0 / 退货率 > 10%，先修产品再投品牌广告 — 否则品牌负面口碑扩散反而害自己。

### 5. **跨季节看 TACOS 而不是单月**

旺季 TACOS 必然上升（竞争激烈），淡季回落。看年度 TACOS 趋势才是真实品牌效率。

---

## 🚫 5 个常见错误

1. **以为加广告预算就能建品牌** — 品牌是产品力 + 内容 + 信任的综合，不是广告堆出来的
2. **追单月销量增长不看品牌** — 月销量靠促销堆，但品牌词没增长 = 没建任何东西
3. **新品立刻大额加广告** — 评分 < 4.0 时 ROI 必然差，浪费钱
4. **大促前才临时抱佛脚** — 旺季流量 30-50% 集中，需要提前 1-2 月备战
5. **跨品类乱布局** — 没建立品类心智的品牌，扩品类只会稀释资源
