# 排名下降应对 SOP（广告 KPI 健康但市场份额流失）

> 适用场景：品线/产品自然排名下降 + 竞品上升 + 广告 ACOS/ROAS 看着没问题
> 配套：[投手周复盘SOP.md](投手周复盘SOP.md)（日常用）+ 本 SOP（应急用）

---

## ⚠️ 核心认知 — 这不是广告问题

**广告 KPI 健康 + 排名下降** = **自然流量份额流失**，不是广告效率问题。

需要的不是"优化广告投放"，而是**用广告补自然流失 + 推动排名回升**。两者目标不同：

| 维度 | 日常优化 | 排名下降时 |
|---|---|---|
| 目标 | 投得准、花得少 | 抢回份额 / 推排名 |
| TACOS | 越低越好 | 容忍短期上升 |
| ACOS | 越低越好 | 容忍短期上升 |
| 关键看 | 广告效率 | 自然订单占比 / 排名 / Sessions |

---

## 🔍 Step 1：先排查根因（80% 不是广告问题）

### 5 类常见根因 — 按概率排序

| 根因 | 怎么查 | 典型信号 |
|---|---|---|
| **1. 库存断过货** ⭐ 最常见 | 历史库存数据，看过去 30 天有无"可售 <14d"窗口 | 缺货 1 周排名掉 30-50 位 |
| **2. 评分恶化** | Amazon 后台 → Voice of Customer / Reviews / 评分趋势 | 平均分降 0.1 排名掉 10 位 |
| **3. 价格不竞争** | Amazon 前台搜核心词 → 看 Top 5 竞品当前售价 | 自家比竞品贵 10%+ |
| **4. Listing 退化** | 自查近 30 天 listing 改动（标题/主图/五点/A+）| 排名掉的时点 = listing 改动时点 |
| **5. 新竞品冲击** | Helium10/JS 看 BSR 趋势 — Top 10 有无新面孔 | 新品冲入挤掉老品 |

### ❗ 关键提醒

广告诊断报告里看不出这 5 类根因 — 必须去 Amazon 后台 + 第三方工具查。

---

## 🎯 Step 2：按根因对症下药

### A. 缺过货 → 补货 + 主动重启排名

```
1. 立即补货到 60d+ 库存
2. 补货后"重启"广告排名:
   - 提高核心词 Bid 20-30%
   - 提 Top of Search Bid Modifier 到 +150%
   - 配 Coupon 10-20% 拉销量速率
3. 持续 2-4 周
```

**TACOS 容忍**：从 10% → 短期 18-20% 是值得的。

### B. 评分恶化 → 先修评论再加投

```
1. 改产品/包装(修差评原因)
2. 主推 Amazon Vine 程序送评
3. 短期不要加广告(转化率低反而浪费)
4. 评分回升后再加投
```

### C. 价格不竞争 → 价格策略 + 差异化

```
1. 跟随定价(调到 Top 3 竞品同档)
2. 促销组合(Coupon + Deal + Prime 专享)
3. 广告主推自家差异化卖点(品牌/包装/保修)
```

### D. Listing 退化 → 立即回滚

```
1. 找到改坏的版本 → 恢复主图/标题
2. 之后任何 listing 改动必须 A/B 测试
```

### E. 新竞品冲击 → 进攻 + 防守组合

**进攻**:
- **SD VCPM 投竞品 ASIN** — 在竞品详情页截流
- **SP PAT 商品定向** — 直接投竞品 Top 3 ASIN
- **SB 视频抢核心词头部** — 用户搜词第一眼看到自家
- **加 Top of Search Bid+** — 抢搜索结果首页第一位

**防守**:
- **本品防御 SD** — 防止自家详情页被竞品占位
- **品牌词全量覆盖** — 不让竞品截流

---

## 🎯 Step 3：投放策略切换 — "效率模式" → "份额模式"

### 战术对照表

| 维度 | 效率模式（默认）| **份额模式**（本期）|
|---|---|---|
| TACOS 目标 | 8-12% | 容忍 15-22% |
| **Top of Search Bid+** | 50-100% | **150-300%** |
| 核心品类词出价 | 健康 CPC 水平 | 提 20-30% |
| 预算分配偏向 | 平均分 SP/SB/SD | **SB Video + SD VCPM 加权**（曝光型）|
| 关键词结构 | Exact 主 | Exact + Phrase 都加投 |
| **Coupon** | 偶尔做 | **持续 10-15%** 拉销量速率 |
| 新品/拓词 | 谨慎测试 | **暂缓**（聚焦保排名）|
| Auto Campaign | 紧密为主 | 加开宽松 Auto 找新流量 |

### 短期可容忍的"广告报告里看着差"

| 信号 | 容忍原因 |
|---|---|
| ACOS 上升 5-10pp | 抢量阶段必然 |
| 单 SKU 短期 0 单 | 抢曝光阶段 |
| Top of Search CPC 上升 | 黄金位置贵但效率最高 |
| 整体 ROAS 短期降到 3-4 | 平时 5+ 但抢量值得 |
| 个别 Campaign Spend 翻倍 | 主投核心词正常现象 |

### 绝对**不能**容忍

| 信号 | 后果 |
|---|---|
| 库存继续降到 30d 内 | 再缺货排名彻底崩 |
| Sessions 持续下降 | 流量端崩盘，加广告也救不回 |
| 评分继续恶化 | 产品力问题，广告是浪费 |
| TACOS 爆到 >30% | 失血过多，需重新评估策略 |

## 📊 Step 4：监控指标切换

### 排名下降阶段每周看的指标（从上往下重要性递减）

```
🔴 主指标（每周必看）
├── 1. 核心词排名（用 Helium10/JS）— 目标 4 周内恢复
├── 2. BSR 趋势 — 目标 4 周内回到下降前 Top 区间
├── 3. Sessions 周环比 — 增长或稳定
└── 4. 自然订单占比 — 从下降 → 回升

🟡 副指标（保持观察）
├── 5. 店CVR — 保持稳定（如降则产品力问题）
├── 6. 评分 — 不能继续下降
└── 7. TACOS — 容忍升，4 周后开始回降

🟢 旁观指标（暂时不是主战场）
├── 8. ACOS / ROAS — 容忍恶化
└── 9. 广告报告里的"🚨 优化重点型"标签 — 暂时忽略
```

## 🛠️ 实操执行清单（按时间）

### 第 1 周（诊断 + 启动）

| Day | 动作 |
|---|---|
| Day 1 | 5 类根因排查（库存/评分/价格/listing/竞品）→ 定位真因 |
| Day 1 | 推送补货工单（如果缺货是因）|
| Day 2 | 启动 Coupon 10-15%（拉销量速率）|
| Day 2 | 提核心词 Bid + Top of Search Bid Modifier |
| Day 3 | 开 SD VCPM 投竞品 ASIN（如新竞品冲击是因）|
| Day 3 | 评论修复启动（Vine / Request a Review）|
| Day 4-7 | 监控 Sessions / BSR 变化 |

### 第 2-3 周（持续优化）

- 看 Sessions 是否回升 — 不回升 = 战术失效，重新评估
- 看核心词排名 — 用工具记录
- TACOS 容忍上升，但 >30% 重新评估

### 第 4 周（评估 + 收尾）

- 排名是否回到下降前水平？
- 如**回升** → 逐步降回"效率模式"参数（Bid Modifier 从 +300% 降回 +100%；Coupon 频次降低）
- 如**未回升** → 重新做根因分析（可能错诊了；可能是产品力问题；可能竞品已重塑品类）

## 💡 经验法则

### 1. "用广告补自然流失"不是万能的

- 如果是**产品力问题**（评分差/价格不竞争/listing 差）→ 广告再加也救不回
- 必须先修产品/listing，再用广告推
- 单纯堆广告预算 = 烧钱无效

### 2. **销量速率 > ACOS** 是这阶段的核心指标

- Amazon 排名算法看的是"过去 24-72 小时销量速率"
- Coupon + 广告组合就是为了在短期拉销量速率 → 触发排名上升
- 这阶段一周 200 单 ACOS 25%，比一周 50 单 ACOS 10%（份额已失） 价值高得多

### 3. 抢排名是"投资"不是"消费"

- 短期 TACOS 18-22% 看起来高
- 但排名上升后，自然订单回来，**未来 6-12 个月 TACOS 会降到 5-8%**
- 这是"投资期 vs 收成期"的不同 — 财务上是 ROI 正向的

### 4. **永远不要在缺货时加广告预算**

- 缺货 + 广告引流 = 用户进来发现无货 → 跳出 → Listing 排名进一步降
- 必须先确保库存 ≥60d 再加投
- 如果是缺货后补回来，补货到位再加投

---

## 🚫 5 个常见错误

1. **看 ACOS 升就紧张要砍预算** — 抢排名期 ACOS 必然升，看 TACOS + 自然占比才是真信号
2. **以为加广告预算就能挽回** — 没解决根因（评分/价格/listing）单纯加预算无效
3. **缺货期还加广告** — 加速排名死亡
4. **同时修多个变量** — 改了价 + 改了图 + 加了 Bid + 上了 Coupon → 不知哪个起作用。**一次只改 1-2 个变量**
5. **2 周看不到效果就放弃** — 排名重建需要 4-8 周，过早收手前功尽弃
