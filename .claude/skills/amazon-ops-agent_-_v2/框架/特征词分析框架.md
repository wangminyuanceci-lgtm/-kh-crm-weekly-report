# 特征词（客户搜索词 + 词根）分析框架

> 配套脚本：[../scripts/search_term_analysis.py](../scripts/search_term_analysis.py)
> 数据源：BulkSheetExport.xlsx 的 SP/SB Search Term Report
> 输出：`<output_dir>/<品线>/客户搜索词分析.xlsx` — 含 7 sheet（首 sheet 是品线基准与诊断标准）

---

## 📐 诊断标准（脚本已自动应用）

### 阈值定义

| 阈值 | 计算公式 | 含义 |
|---|---|---|
| **大盘 ACOS** | sum(Spend) / sum(Sales) | 品线整体广告效率基准 |
| **大盘 CVR** | sum(Orders) / sum(Clicks) | 品线整体转化率 |
| **大盘 CTR** | sum(Clicks) / sum(Impressions) | 品线整体点击率 |
| **ACOS 上限阈值** | 大盘 ACOS + 30pp | 超过此阈值视为"效果较差" |
| **点击量基准** | 1 / 大盘 CVR | 此点击数下若仍 0/低单则数据基础足够下结论 |

### 诊断状态分类

| 标记 | 触发条件 | 含义 | 建议动作 |
|---|---|---|---|
| 🚨 **效果较差** | Click ≥ 点击基准 **且** ACOS > 上限 | 数据基础足且确实差 | 下钻 Top 3-5 烧钱 Campaign 排查根因 |
| ⭐ **表现优秀** | Click ≥ 点击基准 **且** ACOS ≤ 大盘 **且** Orders ≥ 3 | 健康有数据基础 | 加投/扩词（提取为 Exact）|
| **正常** | Click ≥ 点击基准 **且** ACOS 在大盘 ~ 上限之间 | 在大盘范围内 | 维持 |
| **数据不足** | Click < 点击基准 | 样本太小，不下结论 | 等数据积累 |

### CTR 警戒（与 ACOS 独立的副信号）

| 标记 | 触发条件 | 含义 |
|---|---|---|
| ⬇️⬇️ CTR 极低 | < 大盘 × 0.5 | 主图/关键词错配严重 — 优先改 listing |
| ⬇️ CTR 偏低 | < 大盘 | 主图或词匹配度偏弱 |
| ⬆️ CTR 偏高 | > 大盘 × 1.5 | 词命中精准，但要看是否转化（高 CTR + 高 ACOS 异常 — 点了不买）|

## 🔍 "🚨 效果较差"下钻流程

```
Step 1: 在 [按搜索词聚合 sheet] 筛 诊断状态 = 🚨 效果较差
Step 2: 按 Spend 倒序取 Top 3-5（小钱不深抠）
Step 3: 每个搜索词 → 查 [搜索词原始整合 sheet] 找投放 Campaign
        - 若涉及 1-3 个 Campaign:每个都看
        - 若涉及 > 5 个 Campaign:只看 Top 3-5 花费占比大的
Step 4: 识别根因 — 通常是以下 3 类模式之一
```

### 3 类常见根因

| 模式 | 信号 | 修复 |
|---|---|---|
| **同名 Campaign 重复抢竞价** | 同一搜索词涉及多个完全或几乎同名 Campaign | 合并 Campaign 或加 Negative 把流量收到 1 个 |
| **产品-关键词错配** | 搜索词意图 ≠ 投放 Campaign 的产品（如 short barbell 投在 Curl Bar Camp）| 在错配 Campaign 加 Negative |
| **流量大词进入精准产品** | 单一精准产品 Campaign 投在"set/kit"等套装词 | 大词由更合适的产品 Campaign 接 |

## 🔍 词根级诊断（同标准）

词根级 = 把搜索词拆词后按 token 聚合，每个 token 继承所属搜索词的完整指标（行业惯例）。

诊断标准同搜索词级。重点关注：
- **品类错配词根**（如 GYMNASTICS 出现 dumbbell 词根 ACOS 82% — 不卖哑铃却进流量）→ 全品线 Phrase Negative
- **修饰泛词词根**（extra/fitness/pound 等）→ 谨慎，可能误伤精准长尾
- **西语/外语词根**（barra / pesa 等）→ 看是否定位错市场

---

## ⚠️ 5 条铁律（避免单周判断误杀）

1. **Click < 点击基准 = 数据不足** — 单周点击量小的搜索词不下结论
2. **任何 Pause 决策必须看 4-9 周趋势** — 单周高 ACOS 可能是周期性波动
3. **投入低 (单周 Spend < $30) 不动** — 即使本周差，钱小不痛
4. **"⭐ 表现优秀"扩词风险也要控制** — 用小预算 $5/天测试 2 周，不直接大额加投
5. **CTR 警戒只是副信号** — 不能仅凭 CTR 低就 Pause（可能 listing 问题，不是关键词问题）

```
<output_dir>/<品线>/客户搜索词分析.xlsx
├── 0_品线基准与诊断标准      ← 阈值定义
├── 搜索词原始整合             ← Campaign × 搜索词明细

└── 按词根×品线聚合            
```

## 🚀 使用流程

```bash
# 跑单品线
python scripts/search_term_analysis.py --product-lines GYMNASTICS \
  --output-dir ".../<日期>/GYMNASTICS"

# 跑多品线（输出到各自子目录）
python scripts/search_term_analysis.py --product-lines GYMNASTICS,WEIGHTS \
  --output-dir ".../<日期>"
```

跑完后：
1. 打开 xlsx → 看 `0_品线基准与诊断标准` 确认本期阈值
2. 在 `按搜索词聚合` sheet 按"诊断状态"过滤 = 🚨 效果较差 → Top 3-5 下钻
3. 在同 sheet 按"诊断状态"过滤 = ⭐ 表现优秀 → 扩词候选
4. 词根级同样流程

## 💡 跟广告参数调整的关系

| 场景 | 关键词分析价值 |
|---|---|
| 广告参数调整本周不可执行（库存数据失真/单周不可靠）| ⭐ 关键词分析仍可执行扩词（低风险增量动作）|
| 想找烧钱黑洞 | 词根级 🚨 效果较差 + 下钻 Campaign 比直接看 ACOS 高的关键词更准 |
| 想找隐藏机会 | 高 Spend ⭐ 但无 Exact 投放的词 = 扩词最佳候选 |

实战中：当广告参数因数据局限被卡住时，**关键词扩词** 通常是相对低风险高价值的优化方向。
