# AI CRM Operating System 报告模板

使用此模板生成 EDM 周报。三大章节固定不可变，子标题可灵活调整，但骨架结构不变。所有输出使用简体中文。

---

## 报告标题

```
# EDM 周报 — Week {XX}, {YYYY.MM.DD}–{YYYY.MM.DD}
```

标题下方一行元数据：

```
报告周期：{YYYY.MM.DD} – {YYYY.MM.DD} ({时区}) · 币种：{USD} · 品牌：RitFit Sports
```

## 二、数据诊断分析

此章节解释"为什么发生"——每个数据异常都必须有根因假设和置信度标注。

### 2.1 Campaign 深度诊断

#### 2.1.1 素材维度诊断

对每封 Campaign，从以下维度评分（1-5 分）：

| Campaign | 主题行吸引力 | 预览文本 | Hero 区块 | CTA 明确性 | 整体评分 | 邮件预览链接 |
|---|---|---|---|---|---|---|
| | | | | | | [Klaviyo 预览] |

素材诊断评分标准：
- 5 分：元素优秀，可直接复用为最佳实践
- 3 分：可用但无亮点，建议优化
- 1 分：需立即修改，严重拖累表现

邮件预览链接获取方式：
调用 `klaviyo_get_campaign` 获取 `campaign_message_id`，邮件在 Klaviyo UI 中的可分享预览链接格式为：
```
https://www.klaviyo.com/email-editor/{template_id}/edit
```
或通过 Klaviyo 的 Campaign Preview 页面获取可分享链接。

## 输出语言与格式规范

- **语言**：全文中文（品牌名、指标缩写如 CTOR/CVR/RPR/RPM 保留英文）
- **数字格式**：金额 `$12,345`，百分比 `35.2%`，小数百分比 `0.35%`
- **表格**：Markdown 管道表格（飞书兼容）
- **禁止**：
  - Markdown 残留（`**`、`---`、多余空行）
  - 重复货币符号（`$$`）
  - 原始 JSON 或未处理数据
  - AI 套话（"In conclusion"、"It might be worth considering"）
  - 无数据支撑的空泛建议

## 图表规范

报告中以下位置必须附带图表数据（以表格格式提供，由发布系统渲染为可视化）：

1. **1.4A** 近 4 周 Campaign 收入趋势（折线图数据）
2. **1.4B** 近 4 周 Flow 收入趋势（折线图数据）
3. **1.4C** 本周 Campaign 序列 RPM 瀑布（瀑布图/条形图数据）
4. **1.4D** 全账户收入结构（饼图/环形图数据）

发布到飞书时，优先使用飞书画板（lark-whiteboard）渲染趋势图表。
